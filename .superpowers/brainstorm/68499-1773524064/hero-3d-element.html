Hero 3D Element — How to Handle It?

Sutera uses pre-rendered 3D blob animations (animated WebP). For allone, we have options:

A

CSS/SVG Animated Orb

Pure CSS or SVG animated sphere/blob with gradients and rotation. Zero external dependencies, lightweight, looks premium. Can use allone's blue accent as the orb color. Smooth rotation via CSS keyframes.

Lightest option. Works everywhere. Easy to iterate on.

B

Three.js Sphere/Blob

Real-time 3D rotating sphere with custom shader — glass, noise distortion, or metallic material. Interactive (reacts to mouse). More impressive but heavier (~50-100KB JS).

Most impressive. Adds interactivity. Heavier bundle.

C

Spline 3D Scene (like current allone)

Use Spline to design a custom 3D object (sphere, abstract shape) and embed it. Easy to design, interactive, but depends on Spline's CDN and has longer load times.

Easiest to design custom shapes. External dependency. Slower load.

D

Pre-rendered Animation (like Sutera)

Create a 3D blob animation in Blender/After Effects, export as animated WebP or short looping video. Looks 3D but loads like an image. Best performance.

Best perf. Requires external tool to create the asset. Not interactive.
