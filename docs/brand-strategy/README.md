## How to Use This System

**For strategic decisions:** Start with [Strategic Summary](phase-2-strategy/13-strategic-summary.md) or [Strategic Layer](phase-6-governance/30-strategic-layer.md).

**For content creation:** Reference [Verbal Identity](phase-3-identity/14-verbal-identity.md) and [Keywords](phase-3-identity/18-keywords-communication.md).

**For design work:** Reference [Visual Identity](phase-3-identity/15-visual-identity.md) and [UX/UI Guidelines](phase-4-experience/24-ux-ui-guidelines.md).

**For marketing:** Start with [Marketing Strategy](phase-5-activation/25-marketing-strategy.md) and [Social Media](phase-5-activation/27-social-media-strategy.md).

**For brand compliance:** Use [Brand Governance](phase-6-governance/33-brand-governance.md) as the definitive reference.

**For client-facing materials:** Cross-reference [Buyer Personas](phase-2-strategy/11-buyer-personas.md) with [Consumer Journey](phase-4-experience/22-consumer-journey.md).
